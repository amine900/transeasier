Features

GPS Tracking

Using the GPS tracking function in TransEasier, passengers can see arrival updates in real-time, directly from the maps.

Public Transport Schedules

Users can easily access to the public means of transport Schedules: departure and arrival, through the platform, Mobile App or through the panels in the station.

A set of Panels

TransEasier will provide a set of panels in each station connected to the platform and mobile App indicating real-time information.

Maps

Users can easily access to a map service that helps to find the shortest way to his destination with some other information.

Notifications

Our notifications system will offer the users real-time information about arrivals and departures.

Clear indications about time, weather conditions, Traffic

Users can easily get this information, through the platform, Mobile App or through the panels in the station.
